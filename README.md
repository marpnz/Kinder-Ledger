# Kinder-Ledger
Kinder Ladger es una plataforma de iegistro inmutable de Solana, diseñada para casas hogar. Permite digitalizar expedientes infantiles (nombre, historial medico y fechas de ingreso) de forma permanente y segura en blockchain, garantizando que el historial de cada niño sea rastreable, no pueda ser borrado y se mantenga disponible para cualquier cosa
